# Happy obfuscated birthday!
This code calculates my age on my 2026 birthday in a way that is hopefully not entirely obvious. You can obtain, compile and run the code by typing the following in a console window:

```
git clone https://github.com/guyhill/birthday2026.git
cd birthday2026
make -s run
```

But the real challenge is to try to work out how the code works, and from it, how old I will be on my birthday without running it first. 
If you get stuck,take a look at the de-obfuscation document in the `spoilers` directory. I also included some design notes, which are
in effect, the same as the de-obfuscation guideline, but in reverse.

Code only works on Linux.
